## Description ##

Handles ACF JSON data to avoid situational overwrites.

## Installation ##

### Manual Installation ###

1. Upload the entire `/wds-acf-json-juggler` directory to the `/wp-content/plugins/` directory.
2. Activate WDS ACF JSON Juggler through the 'Plugins' menu in WordPress.
